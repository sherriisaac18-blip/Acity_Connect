# Acity Connect

## Project Overview
Acity Connect is a campus-based platform that allows students to buy, sell, and exchange skills and products.

Users can:
- Browse products such as laptops, textbooks, and calculators  
- Explore skills including tutoring, proofreading, and video editing  
- Register through a simple signup form  

This project is a front-end web application built using HTML and CSS.
